  • Siqi’s chosen set of tools

    • task management: orgmode in Emacs & google tasks

    • writing: MS word, scrivener or overleaf, depending on what the collaborators prefer

    • multi-language code editor: VS code

    • data analysis: python, R, or stata, depending on what the collaborators prefer

    • presentation: keynote or jupyter notebook, seldom use latex beamer (but my job talk was on a set of beamer slides)

    • literature management: devonthink3, zotero, reearch rabbit

    • distant access to office computer: teamviewer

    • cloud storage: I maintain a private synology NAS, but also use dropbox and onedrive to suit collaborators’ need

    • webpage annotation: Readwise

    • chrome addon: vimium, onetab

    • window manager: amethyst
